This project is a responsive web page implementation based on the Figma design provided by Gushwork as part of the Web Developer assignment.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Features Implemented

### Responsive Design
The page is fully responsive and works across:
- Desktop
- Tablet
- Mobile devices

### Sticky Header
- Header appears when scrolling beyond the first fold
- Smooth transition animation
- Disappears when scrolling back to the top

### Image Carousel with Zoom
- Interactive image carousel
- Hover zoom preview effect on images
- Smooth hover transitions

### Code Quality
- Semantic HTML structure
- Clean and organized CSS
- Vanilla JavaScript interactions
